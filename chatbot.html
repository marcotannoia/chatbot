WHAT IS THIS PROJECT?

This chatbot is my first serverless project. I created it to learn, experiment and understand how a fully scalable cloud application works. How does it work?

Simply type a message and the bot will respond using AI-generated text.

DIFFICULTIES I FACED & HOW IT WAS BUILT

This project uses:

  • Amazon S3 for hosting the frontend
  • Amazon CloudFront for global HTTPS delivery
  • Amazon Cognito for user authentication, which guarantees secure access
  • AWS API Gateway (HTTP API) to receive chat messages
  • AWS Lambda to run the serverless backend
  • Gemini Free Tier Model for AI responses

I faced many difficulties, especially with finding AI API which was easy to use and free to start with.

I finally settled on Gemini Free Tiar Model, which provides a free tier and a variety of model, it has some limitations , such as a limited number of tokens per request, but for this project it works well.

Another challenge was setting up the AWS services to work together, especially Cognito and API Gateway, and the fact that Cognito requires HTTPS, which meant setting up CloudFront in front of S3.

The frontend was built totally by ChatGPT, I just gave it some instructions and it generated the HTML, CSS and JS code for me. Also I asked ChatGPT to help me with facing some issues I had while creating the overall architecture of the project. Nonetheless, I had to debug and modify the code myself to make everything work as I wanted, since ChatGPT sometimes generates code that doesn't work as expected.

Overall, it was a great learning experience and I'm happy with the result!
